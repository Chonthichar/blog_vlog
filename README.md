# Interactive Blog & Vlog Application

This project presents an interactive blog and vlog application, developed using the Django framework and deployed on Amazon Web Services (AWS). The application's core purpose is to enable users to create and share posts that include both text and multimedia elements. Key features, such as commenting and liking posts, enhance the user experience, making this platform dynamic and engaging.
